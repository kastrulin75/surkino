# SURKINO

Независимый редизайн афиши кино в Сургуте.  
Обычный фронтенд: HTML / CSS / JavaScript.

> Сайт сделан Матвеевым Ефимом в 2026.  
> Это некоммерческий / образовательный / портфолио-редизайн оригинального [surkino.ru](https://surkino.ru/).  
> Автор не претендует на права на контент и товарные знаки правообладателей.

## Запуск

Откройте `index.html` в браузере или поднимите локальный сервер:

```bash
npx serve .
```

## Структура

```
index.html          — главная / афиша
schedule.html       — расписание
films.html          — каталог фильмов
film.html?id=       — карточка фильма
cinemas.html        — кинотеатры
cinema.html?slug=   — страница кинотеатра
css/styles.css      — стили (Liquid Glass)
js/                 — данные и логика
assets/img/         — фото кинотеатров и постеры
```

## Возможности

- Тёмная / светлая тема
- Расписание по дате и кинотеатру
- Каталог фильмов и блок «Скоро»
- Интро при первом заходе
- Плавные переходы между страницами
- Адаптив под мобильные

## Данные

- Кинотеатры и фильмы: `js/data.js`
- Расписание (демо-генерация): `js/schedule.js`
